# Wrapper around aNCA::pkcg03() function. Calls the function with `LIN` scale argument.

Wrapper around aNCA::pkcg03() function. Calls the function with `LIN`
scale argument.

## Usage

``` r
g_pkcg03_lin(data, ...)
```

## Arguments

- data:

  Data to be passed into the plotting function.

- ...:

  Any other parameters to be passed into the plotting function.

## Value

ggplot2 object for pkcg03.
