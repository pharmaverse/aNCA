# Create a new PowerPoint document from a template and add a title slide

Create a new PowerPoint document from a template and add a title slide

## Usage

``` r
create_pptx_doc(path, title, template)
```

## Arguments

- path:

  File path to save the presentation

- title:

  Title for the presentation

- template:

  Path to PowerPoint template file

## Value

rpptx object
