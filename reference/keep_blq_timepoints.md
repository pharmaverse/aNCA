# Helper function for BLQ processing

Helper function for BLQ processing

## Usage

``` r
keep_blq_timepoints(plot_data, xvar, mean_group_var)
```

## Arguments

- plot_data:

  Data frame containing the data to be processed.

- xvar:

  Character string of the variable name for the x-axis.

- mean_group_var:

  Character string specifying the grouping variable to plot by group.

## Value

A data frame with valid timepoints based on BLQ criteria.
