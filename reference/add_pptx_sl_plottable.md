# Add a slide with both a plot and a table

Add a slide with both a plot and a table

## Usage

``` r
add_pptx_sl_plottable(pptx, df, plot)
```

## Arguments

- pptx:

  rpptx object

- df:

  Data frame to show as table

- plot:

  ggplot object to show as plot

## Value

rpptx object with slide added
