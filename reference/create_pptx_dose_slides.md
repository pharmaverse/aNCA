# Create a PowerPoint presentation with dose escalation results, including main and extra figures Adds slides for summary tables, mean plots, line plots, and individual subject results

Create a PowerPoint presentation with dose escalation results, including
main and extra figures Adds slides for summary tables, mean plots, line
plots, and individual subject results

## Usage

``` r
create_pptx_dose_slides(res_dose_slides, path, title, template)
```

## Arguments

- res_dose_slides:

  List of results for each dose group

- path:

  File path to save the presentation

- title:

  Title for the presentation

- template:

  Path to PowerPoint template file

## Value

TRUE (invisible). Writes the PowerPoint file to the specified path
