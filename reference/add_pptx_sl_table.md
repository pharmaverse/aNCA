# Add a slide with a table only

Add a slide with a table only

## Usage

``` r
add_pptx_sl_table(
  pptx,
  df,
  title = "",
  footer = "Click here for individual results"
)
```

## Arguments

- pptx:

  rpptx object

- df:

  Data frame to show as table

- title:

  Title text for the slide

- footer:

  Footer text for the slide

## Value

rpptx object with slide added
