# Apply a single filter condition to a data frame

Apply a single filter condition to a data frame

## Usage

``` r
apply_single_filter(data, column, condition, value, include_na)
```

## Arguments

- data:

  A data frame.

- column:

  Column name to filter on.

- condition:

  Filter condition string.

- value:

  Filter value(s).

- include_na:

  Logical; whether to include NA rows.

## Value

A filtered data frame.
