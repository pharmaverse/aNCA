## Goals for 2026

This roadmap intends to capture the main goals that are planned to be achived in the next year. However, notice that this is just a high-level picture, and specific tasks and issues will be tracked in the GitHub repository and in the (project board)[https://github.com/orgs/pharmaverse/projects/30].

- [] Provide an R script output that fully reproduces the NCA analysis as performed in the Shiny app, including data preprocessing, mapping, NCA settings, and TLG generation.
- [] Allow the user to handle exclusions at the data level for NCA running and PK summary outputs.
- [] Allow the user to handle BLQ data points.
- [] Allow the sparse data handling option.
- [] Include Tables in the TLG section, as well as new Graphs and Listings.
- [] Provide draft reports for PK submission
- [] Allow handling of TK and oligo data in the App.
